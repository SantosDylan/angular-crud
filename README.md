# ðŸ”¨ Getting Started

## Quick cloning

```
$ git clone git@gitlab.com:SantosDylan/angular-crud.git
```

## Setup

```
$ cd angular-crud
$ npm install
```

## Building

```
$ json-server db.json -m ./node_modules/json-server-auth -r routes.json
$ ng serve
```
